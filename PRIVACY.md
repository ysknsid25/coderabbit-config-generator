# Privacy Policy

Last updated: 2026-07-16

## Summary

**This application does not collect or store any information you enter or upload.**

CodeRabbit Config Generator runs entirely in your browser. The form data you fill in and the generated YAML configuration never leave your device — nothing is sent to, or stored on, any server.

## Details

- **No data collection**: The app does not transmit form input, uploaded files, or generated configuration to any backend, analytics service, or third party.
- **No storage**: No cookies, local storage, or server-side database is used to persist your input across sessions.
- **Static hosting only**: The app is a static site (HTML/CSS/JS) served as-is. There is no backend server processing your data.
- **Open source**: The full source code is publicly available at [github.com/ysknsid25/coderabbit-config-generator](https://github.com/ysknsid25/coderabbit-config-generator). You are welcome to review it yourself to verify these claims, or build and run the app locally.

## Third-Party Services

This site may use privacy-respecting analytics (e.g., page-view counts) to understand aggregate traffic. Such analytics do not receive the content of the form you fill in — only standard, anonymized page-visit metadata.

## Changes to This Policy

If this policy changes, the update will be reflected in this file with a new "Last updated" date, and the change history will be visible in the project's public Git history.
